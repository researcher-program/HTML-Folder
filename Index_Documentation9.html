HTML ~ The Citation Element_Tag { All }

The <cite> HTML element is used to mark up the title of a creative work. The reference may be in an abbreviated form according to context-appropriate conventions related to citation metadata.

More information can be found in [ISO-0000].

HTML ~ The Inline Code Element_Tag { All }

The <code> HTML element displays its contents styled in a fashion intended to indicate that the text is a short fragment of computer code. By default, the content text is displayed using the user agent's default monospace font.

The push() method adds one or more elements to the end of an array and returns the new length of the array.

The function selectAll() highlights all the text in the input field so the user can, for example, copy or delete the text.

HTML ~ The Deleted Text Element_Tag { All }

The < del > HTML element represents a range of text that has been deleted from a document. This can be used when rendering "track changes" or source code diff information, for example. The < ins > element can be used for the opposite purpose: to indicate text that has been added to the document.
